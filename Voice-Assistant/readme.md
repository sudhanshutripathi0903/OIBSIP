# Voice Assistant

This is a simple Python-based Voice Assistant developed for the Oasis Infobyte Internship (OIBSIP).  
It uses text-to-speech to respond to basic commands like greetings, name, and exit.

## Features
- Greets the user
- Speaks responses using pyttsx3
- Accepts typed commands
- Responds to basic questions
- Can exit the assistant

## Technologies Used
- Python 3
- pyttsx3 (Text-to-Speech Library)

## How to Run
1. Install pyttsx3:
   pip install pyttsx3

2. Run the program:
   python assistant.py

3. Example Commands:
   - hello
   - your name
   - time
   - exit

## Output Example
When the user types `hello`, the assistant replies:
"Hello Sudhanshu, how are you?"
